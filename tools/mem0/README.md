# Mem0

**Repository:** [https://github.com/mem0ai/mem0](https://github.com/mem0ai/mem0)

## Overview
Mem0 is a universal memory layer for AI Agents, enabling them to maintain long-term knowledge and context across sessions.

## Key Features
- **Universal Memory:** usable across different agent frameworks.
- **Long-term Knowledge:** Persist user preferences and facts.
- **Self-Hosted & Cloud Options:** Flexible deployment.

## Use Cases
- Personalized learning assistants.
- Context-aware customer support.
- Long-running research agents.

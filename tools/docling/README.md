# Docling

**Repository:** [https://github.com/docling-project/docling](https://github.com/docling-project/docling)

## Overview
Docling is a tool to get your documents ready for Generative AI. It simplifies the ingestion and parsing of various document formats into a structure usable by LLMs.

## Key Features
- **Document Parsing:** Handles PDF, DOCX, and other formats.
- **GenAI Ready:** targeted output formats for RAG and agentic workflows.
- **CLI & Python SDK:** Flexible usage.

## Installation
```bash
pip install docling
```

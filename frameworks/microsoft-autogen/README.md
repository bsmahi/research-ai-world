**Microsoft AutoGen** is a framework that enables the development of LLM applications using multiple agents that can converse with each other to solve tasks. AutoGen agents are customizable, conversable, and seamlessly allow human participation.

- **Multi-Agent Conversation**: Agents can communicate with each other to solve complex workflows.
- **Customizable Agents**: Create agents backed by LLMs, tools, or humans.
- **Flexible Conversation Patterns**: Supports diverse conversation patterns for autonomy.
- **Enhanced LLM Inference**: Utilities to optimize LLM usage and cost (e.g., caching, error handling).

# Google Agent Development Kit (ADK)

**Google Agent Development Kit (ADK)** is a framework designed to simplify the development, testing, and deployment of AI agents. It provides a modular approach to building agents that can coordinate tasks and workflows, optimized for the Google Cloud ecosystem and Gemini models.

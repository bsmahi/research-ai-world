# MCP Updates: AWS & Google Support

## Overview
Major cloud providers are officially supporting the Model Context Protocol (MCP), solidifying its position as a standard for agentic architectures.

## AWS Support
**Source:** [AWS Open Source Blog](https://aws.amazon.com/blogs/opensource/shaping-the-future-of-mcp-aws-commitment-and-vision/)

AWS has committed to the long-term success of MCP, integrating it into their agentic vision. This move under the Linux Foundation ensures an open ecosystem.

## Google Support
**Source:** [Google Cloud Blog](https://cloud.google.com/blog/products/ai-machine-learning/announcing-official-mcp-support-for-google-services)

Google announced official MCP support for Google and Google Cloud services. This allows developers to turn existing APIs into secure and governed agentic tools using Apigee and other Google Cloud services.
